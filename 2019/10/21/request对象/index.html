  1. 基本的网页响应形式

用户端界面发送请求–>响应端界面

  1. request应用实例

  • 首先新建一个文件名为login.jsp文件，源码如下

      <%@pagecontentType="text/html;charset=UTF-8"language="java"%>
  <html>
  <head>
  <title>getRequest.jsp</title>
  </head>
  <body>
  <div style="text-align:center">
  <forma ction="响应的界面名称"method="响应方式有两种1.post2.get"一般为post>

  <tr>
  <td>用户名称</td>
  <td><input name="usename"type="text"></td>
  </tr>
  <tr>
  <td>用户密码</td>
  <td><input name="password"type="password"></td>
  </tr>
  <tr>
  <td><input type="submit"value="登录"></td>
  </tr>

  </form>
  </div>
  </body>
  </html>

  • 如上文中的<input>属性中的name就是响应端能够得到的数据的名称，type则为该数据的数据类型（如：设置为”password”，用户密码输入时的内容将用”*”代替显示）其中我们把用户输入的数据放入

    标签中，这样当用户点击type为submit按钮界面就会跳转到响应的页面中。

  1. 取得用户输入的数据

  • 首先新建一个名为”getInfo.jsp”JSP文件,源码如下：
     <%@pagecontentType="text/html;charset=UTF-8"language="java"%>
 <html>
 <head>
 <title>getInfo.jsp</title>
 </head>
 <body>
 <%
 String name=request.getParameter("usename");
 String password=request.getParameter("password");
 %>
 </body>
 </html>

```

  • 这样用户的输入数据就保存在name和password中了，你就可以利用用户给出的数据执行一些操作。

  • 关于数据的存储还有创建对象的方式，详细会在下次内容中讲解。
